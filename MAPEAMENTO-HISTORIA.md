# Mapeamento — História (8º, 9º e 1º colegial)

Recorte do acervo inicial do **Guia do Conhecimento**.  
Organização no estilo Toda Matéria: matéria → unidades/tópicos → conteúdo.

---

## 1. Como vamos montar

| Etapa | O que faz |
|-------|-----------|
| A | Usar a **BNCC** como grade oficial de tópicos (8º e 9º) |
| B | Para o **1º colegial**, usar a área Ciências Humanas da BNCC-EM + progressão típica de História (séculos XX–XXI / aprofundamento) |
| C | Buscar **texto aberto** nas fontes da seção 3 e adaptar para o app |
| D | Em cada conteúdo no Firebase: `source`, `sourceUrl`, `license` |

A BNCC **não é o texto de estudo** — é o **índice**. O texto vem das fontes abertas.

### 2.3 — 1º colegial (`levels: ["1medio"]`)

Na BNCC do Ensino Médio, História entra em **Ciências Humanas e Sociais Aplicadas** (sem lista fechada por série).  
Para o app, usamos um **recorte prático de 1º ano** (aprofundamento + revisão do que cai no começo do médio):

| Bloco | Tópicos sugeridos |
|-------|-------------------|
| Revisão / ponte | Independências, Império, República Velha (síntese) |
| Mundo contemporâneo | Industrialização, imperialismo, 1ª Guerra, Revolução Russa |
| Entre guerras | Crise de 1929, fascismos, 2ª Guerra |
| Brasil no século XX | Era Vargas, populismo, desenvolvimentismo |
| Guerra Fria e Brasil | Guerra Fria, 1964, ditadura, redemocratização |
| Cidadania | Constituição de 1988, direitos humanos, desigualdades |

Muitos tópicos **cruzam** com o 9º ano — no Firebase o mesmo `topic`/`content` pode ter `levels: ["9ano","1medio"]` quando fizer sentido (evitar duplicar texto).

## 4. Estratégia de preenchimento do Firebase

1. Criar `subject`: História  
2. Criar `topics` conforme as tabelas do §2 (começar pelo **9º**, já que é o ano dela)  
3. Para cada tópico, buscar 1 fonte aberta (Wikipedia CC BY-SA **ou** eduCAPES/Domínio Público)  
4. Gerar/adaptar um **resumo escolar** (corpo do `content.body`) + metadados de fonte  
5. Opcional: 3–5 exercícios por tópico (quando a fonte tiver ou criarmos depois)  
6. Marcar `levels` corretos; reutilizar conteúdo entre 9º e 1º médio quando for o mesmo tema  

### Ordem sugerida do primeiro lote
1. 9º — República / Vargas / Ditadura / Constituição 1988 (mais útil agora)  
2. 8º — Independência e Império  
3. 1º médio — blocos de aprofundamento + tags compartilhadas
